- [Ubuntu + Openbox]
- [Ubuntu + dwm](https://github.com/cheomanigua/config/tree/master/suckless/ubuntu)
- [FreeBSD + dwm](https://github.com/cheomanigua/config/tree/master/suckless/freebsd)
